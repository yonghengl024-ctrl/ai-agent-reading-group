# AI-Agent 教材共学分享

> 本页面为学习交流资料导航，非教材官方网站或官方课程页面。

## 快速入口

| 入口 | 内容 |
|---|---|
| 章节目录 | 按章节浏览汇报资料 |
| 视频链接 | 每章 `videos.md` 中维护外部视频链接 |
| PPT/PDF | 每章 `slides/` 目录中维护汇报材料 |
| 学习笔记 | 每章 `notes.md` 中维护要点、问题和延伸阅读 |
