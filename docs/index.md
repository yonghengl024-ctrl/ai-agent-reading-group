# AI-Agent 使用介绍与实践资料

本页面用于展示 AI-Agent 使用介绍与实践资料，包括章节教材链接、讲解视频、PPT/PDF 和实践笔记。

> 本页面为 AI-Agent 使用资料导航，非教材官方网站或官方课程页面。

## 项目说明

本资料库将教材链接、章节讲解视频、PPT/PDF 和实践笔记统一整理到 GitHub 中，帮助读者按主题了解 AI-Agent 的基础概念、工具配置、提示词工程、Skills、多智能体、Hooks、知识库和典型应用场景。教材正文请以原教材网站为准，本页面主要作为资料导航。

## 快速入口

| 入口 | 内容 |
|---|---|
| 章节目录 | 按主题浏览资料和教材链接 |
| 视频链接 | 每章 `videos.md` 中维护外部视频链接 |
| PPT/PDF | 每章 `slides/` 目录中维护讲解材料 |
| 学习笔记 | 每章 `notes.md` 中维护要点、问题和延伸阅读 |
