## 论文写作总规范

1. **学术定位**
   - 按中国博士学位论文标准写作，强调问题意识、理论贡献、识别策略、经验证据与政策含义。
   - 写作应围绕“企业选址—城市规模分布—空间经济结构/城市体系”的主线展开，避免泛泛而谈。

2. **语言风格**
   - 使用正式、克制、逻辑清晰的中文学术表达。
   - 避免口语化、宣传式、绝对化表述；减少“显著提升”“重要意义”等空泛句式。
   - 每段应围绕一个中心论点展开，优先采用“问题—机制—证据—小结”的段落结构。

3. **事实、文献与数据约束**
   - 不得虚构参考文献、作者、年份、期刊、数据来源、回归结果或统计显著性。
   - 如果缺少文献或数据依据，应明确标注“需补充文献/需核验数据/需根据实证结果填写”。
   - 涉及文献时，优先使用真实可核验的经典文献与近年高质量研究；引用格式以后续学校要求为准，默认可按 GB/T 7714 或学校模板整理。

4. **实证写作规范**
   - 区分事实描述、相关性分析、因果识别与机制解释。
   - 写实证结果时必须包含：模型设定、变量定义、样本范围、固定效应、标准误处理、控制变量、经济含义与统计显著性解释。
   - 不得在没有识别设计支撑时使用“导致”“因果影响”等强因果措辞。

5. **数据与可复现规范**
   - 原始数据、处理后数据、代码、输出表格和图形应分层保存；不得直接覆盖原始数据。
   - 变量构造需保留定义、口径、单位、缺失值处理、缩尾/标准化方法与样本筛选规则。
   - 所有实证表格应能追溯到对应脚本、样本和模型设定。

6. **章节一致性**
   - 全文核心概念、变量名称、研究假设、模型编号、表图编号、样本口径应保持一致。
   - 修改某一章时，应检查其对摘要、引言、理论假设、实证结果和结论的连锁影响。

## 建议的论文结构

可按以下博士论文结构组织，具体以学校模板和导师意见为准：

1. 绪论：研究背景、问题提出、研究意义、研究内容、研究方法、创新点、结构安排。
2. 文献综述：企业选址、城市规模分布、集聚经济、空间均衡、城市体系、制度与政策因素等文献。
3. 理论基础与机制分析：核心概念、理论框架、企业选址机制、城市规模演化机制、研究假设。
4. 数据来源、变量构建与事实描述：样本、数据来源、变量定义、数据清洗、描述性统计、空间分布事实。
5. 实证模型与基准结果：计量模型、识别策略、基准回归、结果解释。
6. 稳健性检验与异质性分析：替代指标、样本调整、模型调整、内生性处理、分地区/行业/城市等级分析。
7. 机制检验与拓展分析：中介机制、空间溢出、城市网络、政策冲击或动态效应等。
8. 结论与政策启示：主要发现、理论贡献、政策建议、研究不足与未来展望。

## 自定义 `.skill` 文件使用说明

本目录包含一组面向博士论文写作的 `.skill` 提示文件。使用时可把对应文件内容作为 Claude Code 的任务说明，或在提问时明确“按某某 skill 执行”。

- `大纲生成.skill`：生成或重构整篇博士论文大纲。
- `第1章_绪论与研究背景写作.skill`：写作研究背景、问题提出与研究意义。
- `第2章_文献综述写作.skill`：整理文献综述和研究述评。
- `第3章_理论模型与研究假设写作.skill`：构建理论机制、模型与假设。
- `第4章_数据处理与变量构建写作.skill`：梳理数据来源、变量定义与描述统计。
- `第5章_实证分析写作.skill`：撰写计量模型、基准回归与结果解释。
- `第6章_稳健性与异质性分析写作.skill`：设计稳健性、内生性和异质性分析。
- `第7章_机制检验与拓展分析写作.skill`：撰写机制检验、空间溢出和拓展分析。
- `第8章_结论与政策启示写作.skill`：凝练结论、贡献、政策启示与不足。
- `全文润色与一致性检查.skill`：检查全文逻辑、术语、编号、表述和学术规范。

## 与 Claude Code 协作时的优先级

1. 先确认任务属于：大纲设计、章节写作、文献整理、理论建模、数据实证、润色校对或格式检查。
2. 写作前先询问或读取已有材料，避免重写已经确定的导师意见、研究假设或实证结果。
3. 对缺失信息使用占位符，不得编造。
4. 输出内容优先采用可直接粘贴进论文的正式文本；必要时附“修改说明”或“待补充材料清单”。
5. 任何涉及数据、模型结果、图表和参考文献的内容，都应标注其依据或待核验状态。
