# 后续专题分享

本目录用于收录第一期《AI-Agent》教材学习完成后的其他 AI-Agent 相关分享内容。

`chapters/` 保留为第一期教材导读的 23 章资料；后续专题放在 `topics/`，避免不同来源、不同形式的资料混在同一个章节体系中。

## 建议专题方向

- AI-Agent 工具实操和工作流模板
- 教学、科研、办公、金融、知识管理等场景案例
- 新工具、新框架或重要论文分享
- 提示词、规则文件、Skills、Hooks 和自动化方案
- 多智能体系统设计、评估与部署经验

## 推荐目录结构

```text
topics/topic-name/
├── README.md
├── videos.md
├── slides/
└── examples/
```

其中：

- `README.md`：专题简介、分享人、资料入口、适用场景和引用来源
- `videos.md`：外部视频链接
- `slides/`：PPT、PDF 或 HTML 演示文稿
- `examples/`：可公开的代码、配置、规则文件、示例数据或工作流说明

## 命名建议

专题目录名建议稳定、简短，优先使用英文或拼音：

```text
topics/agent-office-workflow/
topics/research-agent-tools/
topics/multi-agent-cases/
```
