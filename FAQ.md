# 常见问题

## 为什么不直接上传视频到 GitHub？

GitHub 仓库不适合存放大视频文件。视频会显著增加仓库体积，影响克隆、浏览和维护。建议将视频上传到 Bilibili、腾讯会议回放、百度网盘、阿里云盘或对象存储服务，然后在 `videos.md` 中维护链接。

## PPT 应该上传 `.pptx` 还是 `.pdf`？

建议优先上传 PDF，便于在线预览和长期保存。如果讲解人允许二次编辑，可以同时上传 `.pptx`。

## 资料可以公开吗？

公开前应确认讲解人授权，并检查是否包含学生信息、会议聊天记录、未授权图片、未公开研究材料或个人联系方式。

## 这个仓库和教材官网是什么关系？

本仓库是 AI-Agent 使用介绍与实践资料整理项目，非教材官方网站或官方课程仓库。教材内容请以原网站为准。

## 如何补充一个新章节？

复制已有章节目录结构，创建：

```text
chapters/chapter-XX/
├── README.md
├── videos.md
├── slides/
└── examples/
```

然后在根目录 `README.md` 和 `docs/index.md` 中加入该章节入口。

## 教材学完以后，新的分享内容放在哪里？

不建议继续塞进 `chapters/`，因为 `chapters/` 已经代表第一期 23 章教材导读。后续 AI-Agent 专题分享建议放到 `topics/`，每个专题一个独立目录，并在 `topics/README.md`、根目录 `README.md` 和 `docs/index.md` 中补充入口。
